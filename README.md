Jump Planner
============

This is a small thing i wrote for fun. Its an implementation of the A\* algorithm to calculate jump paths in the Eve Online universe.

Requirements
------------

Postgresql database with the data dump loaded


Compilation requirements
------------------------

* libpq (8.4)
* gsoap (2.7)
